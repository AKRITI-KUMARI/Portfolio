# 🌐 Personal Portfolio Website

A responsive and visually appealing portfolio website built using **HTML**, **CSS**, and **JavaScript** to showcase my projects, skills, and professional journey.

---

## 🛠️ Tech Stack

- **HTML5** – Semantic structure and accessibility
- **CSS3** – Responsive design, animations, and layout styling
- **JavaScript (ES6)** – Interactive elements and DOM manipulation

---

## ✨ Features

- 🎯 Clean and modern UI
- 📱 Fully responsive across devices
- 🧠 Interactive navigation and smooth scrolling
- 🖼️ Project showcase with images and descriptions
- 📬 Contact form (can be integrated with Formspree or EmailJS)
- Toggle button for Light and Dark Mode
